# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-07

### Added

- Initial release of the Terraform GCP Cloud Run module
- Cloud Run v2 service resources with full configuration
- Cloud Run v2 job resources with task and retry management
- VPC Access Connector for private networking
- IAM bindings for unauthenticated access control
- Secret Manager integration for environment variables
- Basic and complete usage examples
